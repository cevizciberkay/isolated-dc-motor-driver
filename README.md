# Isolated DC Motor Driver

A compact isolated brushed DC motor driver designed in **KiCad**.

This project demonstrates PCB design practices for motor control applications, including power routing, isolation, connector placement, and component organization.

---

## Overview

This PCB is intended to drive a brushed DC motor while providing electrical isolation between the control circuitry and the power stage.

The design was created as part of my hardware engineering portfolio to improve my PCB design skills for embedded systems and motor control applications.

---

## Features

- Electrical isolation between control and power circuits
- Compact 2-layer PCB layout
- Screw terminal connections
- PWM motor control support
- Easy integration with STM32 or other microcontrollers
- Designed using KiCad
