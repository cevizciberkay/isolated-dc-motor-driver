# Isolated DC Motor Driver v1.0

A simple DC motor control circuit with galvanic isolation using optocouplers.

## Features
- Full galvanic isolation (protects Arduino)
- Forward and Reverse motor control
- Flyback diode for spike protection
- 5V control side / 12V motor side

## Components
- Arduino
- 2x Optocoupler (4N35 or PC817)
- L298N Motor Driver
- 1N4007 Flyback Diode
